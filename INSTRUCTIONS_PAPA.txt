=== GÉNÉRATEUR DE FACTURES - INSTRUCTIONS ===

Bonjour Papa,

Après plusieurs tentatives pour faire fonctionner la version avec fenêtres,
j'ai créé une version simplifiée qui fonctionne dans le Terminal.

FICHIERS IMPORTANTS :
- MODELE FACTURE GLOBAL SOLUTIONS A REMPLIR.pdf (template requis)
- simple_invoice.py (le générateur de factures)
- Lancer_Factures_Simple.command (pour lancer facilement)

COMMENT UTILISER :

1. Assure-toi que tous les fichiers sont dans le même dossier

2. Double-clique sur "Lancer_Factures_Simple.command"
   (Si ça ne marche pas, fais clic-droit → Ouvrir avec → Terminal)

3. Le programme va s'ouvrir dans le Terminal et te demander :
   - Numéro de facture
   - Date (appuie sur Entrée pour aujourd'hui)
   - Informations client
   - Informations projet
   - Articles (jusqu'à 4 articles)
   - Totaux et TVA

4. À la fin, il génère automatiquement le PDF

AVANTAGES DE CETTE VERSION :
✅ Pas de problèmes avec les fenêtres Mac
✅ Fonctionne sur ton système macOS 12.7.6
✅ Calculs automatiques des totaux
✅ Génère le PDF directement
✅ Interface simple en français

NOTES :
- Pour arrêter l'ajout d'articles, laisse le "Libellé" vide
- Les calculs se font automatiquement
- Le PDF sera créé dans le même dossier

Si tu as des problèmes, appelle-moi !

Bon travail avec tes factures,
Ton fils
