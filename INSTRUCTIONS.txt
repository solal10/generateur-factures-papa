============================================
INSTRUCTIONS POUR LANCER L'APPLICATION
============================================

MÉTHODE 1 - Application (Recommandée):
---------------------------------------
1. Double-cliquez sur "Générateur Factures" (icône d'application)
2. L'application se lancera automatiquement

MÉTHODE 2 - Script de lancement:
---------------------------------
1. Double-cliquez sur "Lancer_Factures.command"
2. Si macOS demande l'autorisation, cliquez sur "Ouvrir"
3. L'application se lancera dans une fenêtre

POUR CRÉER UN RACCOURCI SUR LE BUREAU:
----------------------------------------
1. Faites glisser "Générateur Factures" sur le Bureau tout en maintenant Option+Cmd
2. Ou faites un clic droit sur "Générateur Factures" > "Créer un alias"
3. Déplacez l'alias sur le Bureau

UTILISATION DE L'APPLICATION:
-----------------------------
1. Remplissez les champs de la facture
2. Cliquez sur "Générer la Facture PDF"
3. Le PDF sera créé dans le même dossier

EN CAS DE PROBLÈME:
--------------------
- Assurez-vous que tous les fichiers sont dans le même dossier
- Le fichier "MODELE FACTURE GLOBAL SOLUTIONS A REMPLIR.pdf" doit être présent
- Contactez Solal si l'application ne se lance pas
